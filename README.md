# oura_health

[![Flutter](https://img.shields.io/badge/-Flutter-02569B?logo=flutter&logoColor=fff)](https://flutter.dev/)
[![Android](https://img.shields.io/badge/Android-3DDC84?logo=android&logoColor=white)](https://play.google.com/store) [![iOS](https://img.shields.io/badge/iOS-000000?logo=ios&logoColor=white)](https://apps.apple.com/)

A minimal Flutter app for Android & iOS
- Export sleep data from [membership.ouraring.com](https://membership.ouraring.com/data-export)
- Import sleep data to [Google Health Connect](https://developer.android.com/health-and-fitness/guides/health-connect) or [Apple Health](https://www.apple.com/health/)
- Launch [Google Fit](https://www.google.com/fit/) or [Apple Health](https://www.apple.com/health/)
